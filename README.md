# shiptrace

The missing layer between any AI coding/working agent and whatever counts as "shipped" in your domain. A local-first observability layer that records what your AI agents did, joins those events to what actually landed, and surfaces the metrics nobody else publishes — starting with **sessions-to-ship**.

Status: v0.1, in active development.
